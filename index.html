What's inside

Every part of the launch pipeline, handled.

No padding here — this is what's actually wired up in the app, end to end.

01

Microsoft sign-in

Sign in with your real Microsoft account. Tokens refresh automatically so you're not re-logging in every session.

02

Instance manager

Create, rename, re-icon, and delete isolated instances. Each one keeps its own mods, resource packs, and Java choice.

03

Modrinth browser

Search and install mods, modpacks, and resource packs straight from Modrinth, filtered by loader and game version.

04

Automatic Java

Zudo detects installed Java runtimes, and downloads the correct one for your version if it's missing — no manual JDK hunting.

05

Built-in file manager

Browse, create, rename, and delete files and folders inside any instance without leaving the launcher.

06

Mods & resource packs

Toggle mods on or off, check for updates against Modrinth, and reorder your resource pack stack with drag-and-drop.

07

Skin & cape editor

Search and apply skins by player name, upload your own, switch between Classic and Slim arm models, and manage capes — all rendered live.

08

Custom backgrounds

Swap the launcher's backdrop between bundled art or your own uploaded image.

09

Discord Rich Presence

Shows what you're doing and for how long, right in your Discord profile, the moment you launch.

10

Modpack installer

Drop in a modpack and Zudo unpacks it into a ready-to-launch instance automatically.

11

Live console & auto-update

Watch real launch output as it happens, and let Zudo keep itself current — updates download and install in the background.

12

System diagnostics

Pulls your system info up front, so RAM allocation and Java selection are based on what your machine can actually handle.
